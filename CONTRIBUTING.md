# Contributing

Thanks for taking the time to look at this project.

## Getting set up

1. Clone the repo.
2. Run `npm install`.
3. Load `extension/` as an unpacked extension in Chrome, or load `extension/manifest.json` as a temporary add-on in Firefox.

## Workflow

- Branch from `main`.
- Keep each commit focused on a single change.
- Use British English in user-visible copy (organise, colour, behaviour).
- Run `npm run lint` and `npm run format` before pushing.
- Open a pull request and fill in the template.

## Reporting bugs

Use the bug report template. Include the browser version, the page you were on (without sensitive details), and steps to reproduce.

## Code of Conduct

This project follows the [Contributor Covenant](CODE_OF_CONDUCT.md). By participating you agree to abide by it.
