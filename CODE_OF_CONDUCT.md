# Code of Conduct

This project is small and informal. The expectations are simple:

- Be respectful and patient with other people.
- Assume good faith.
- Keep things on topic: bug reports, code, ideas about the project.
- No harassment, slurs or personal attacks. Disagreeing about technical things is fine; disagreeing with someone's identity is not.

If something goes sideways, email **hlisle@protonmail.com** and the maintainer will sort it out.
